# 🌡️ Temperature Converter (Cloud Project)

## 📌 Project Name

BCS601-4MW23CS086

## 📖 Description

This project is a simple web application that converts temperature between **Celsius and Fahrenheit**.
It is developed using **HTML and JavaScript** and deployed on a cloud platform.

---

## 🚀 Features

* Convert Celsius → Fahrenheit
* Convert Fahrenheit → Celsius
* Instant result display
* Simple and user-friendly interface

---

## 🛠️ Technologies Used

* HTML
* JavaScript
* GitHub (Version Control)
* GitHub Pages (Cloud Hosting)

## ⚙️ How It Works

1. User enters a temperature value
2. Selects conversion type
3. Clicks on "Convert" button
4. JavaScript performs calculation
5. Result is displayed instantly

---

## 🧠 Formula Used

* Celsius → Fahrenheit: `(C × 9/5) + 32`
* Fahrenheit → Celsius: `(F − 32) × 5/9`
